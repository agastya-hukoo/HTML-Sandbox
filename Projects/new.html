About Carbon

Carbon is an intuitive and beginner-friendly HTML IDE designed to simplify HTML programming. It features a live preview, syntax highlighting, auto-completion, error detection, and more to enhance your coding experience.

Supported Devices

Currently available for Windows x64 and ARM. Future support for macOS and Linux is planned.

Technical Details

Carbon is built using C#, WinForms, and CefSharp for a seamless and efficient development experience. It offers a modern and customizable interface, with features like theme customization, file management, and a JavaScript console to cater to both beginners and experienced developers.
